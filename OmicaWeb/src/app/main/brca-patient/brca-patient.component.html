Main flow of the project.


PATRIC genomes were saved in Fasta format (file.fasta), divided in many contigs of different lengths. For this research, the genomes were joined in one single string. To read and analyze the genomes, it was decided to read the original genome and the genomeâ€™s complement. Seven Machine Learning models were tested for both bacteria: Random Forest, Adaboost, Xgboost, Decision Trees, Bagging Classifier, Support Vector Machine, KNN. For the seven models I used two training and testing partitions:

  • 70% of the data for training, 30% of the data for testing.
  • 80% of the data for training, 20% of the data for testing.

Select K-best

To score those k-mers with a higher significance to the resistant genomes, I used the SelectKBest function from scikit-learn. The more significant k-mers will be used to find carbapenem (meropenem and imipenem) resistance patterns using the National Center for Biotechnology Information (NCBI) Basic Local Alignment Search Tool BLAST and NCBI GenBank.


Dynammic Programming

After obtaining the most significant k-mer sequences for each model, I used a Dynamic Programming algorithm for finding the k-mer modifications over the reference genome. For each k-mer length, I used different thresholds to validate if the k-mer was or was not within the reference genome.
